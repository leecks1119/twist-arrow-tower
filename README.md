# Twist Arrow Tower

Mobile-first 3D puzzle prototype built with React, Vite, TypeScript, React Three Fiber, Capacitor, and AdMob test wiring.

## Links

- Live test: https://twist-arrow-tower.vercel.app
- GitHub: https://github.com/leecks1119/twist-arrow-tower

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run build
npm run lint
```

## Android Debug APK

```bash
npm run android:build:debug
```

APK output:

```text
android/app/build/outputs/apk/debug/app-debug.apk
```
